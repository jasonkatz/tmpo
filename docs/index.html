tmpo

Autonomous software delivery pipeline.

Describe a task, point it at a repo, and tmpo orchestrates AI agents to plan, implement, test, review, and ship a pull request.

Install
curl -fsSL https://tmpo.sh/install | bash
  • Planner agent reads the codebase and writes a structured proposal
  • Dev agent implements the proposal, commits, and opens a PR
  • Review agent evaluates the diff against the proposal
  • E2E agent runs user journeys and produces evidence
  • Verifier checks evidence against acceptance criteria
  • Failures loop back to dev with context — until all stages pass
